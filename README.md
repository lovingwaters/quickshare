# QuickShare

QuickShare is a lightweight Android application designed for fast and secure sharing of personal information.
Users can store details such as their name, address, phone number, email, and a custom note â€” all displayed in collapsible sections for privacy. With a single tap, any field (or all expanded fields) can be shared via Android's built-in sharing system.

QuickShare is built for simplicity, privacy, and speed.


# Features

Collapsible Information Fields
Each section (Name, Address, Phone, Email, and Shared Note) can be expanded or collapsed with a tap, keeping sensitive information hidden until needed.

Persistent Data Storage
All information is saved locally on the device and restored on the next launch.

Share Buttons

Share individual fields

Share all expanded fields at once

Built-in Android share sheet support

QR Code Generation
A dynamic QR code updates automatically based on the fields that are currently expanded.

Dark Mode Toggle
Users can switch between Light and Dark Mode via the in-app menu.



# Tech Stack

Language: Java

UI Framework: Android View System (XML layouts)

Persistence: SharedPreferences

Minimum SDK: Android 7.0 (API 24)

Target SDK: Latest available

Build Tools: Android Studio / Gradle
